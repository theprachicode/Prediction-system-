# Prediction System

A Flask application for extracting clinical data from PDF reports and predicting peripheral neuropathy risk.

## Structure

- `backend/` — Flask application package
- `data/` — source dataset files
- `models/` — trained ML model artifacts
- `uploads/` — uploaded PDFs
- `templates/` — HTML templates
- `static/` — CSS and JavaScript files
- `database/` — database files (ignored by git)

## Run

```bash
pip install -r requirements.txt
python run.py
```
